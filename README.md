# Joke API Express App

A simple Node.js and Express app that fetches random jokes from the JokeAPI and displays them using EJS.

## Features

- Choose a joke category
- Choose between single jokes and two-part jokes
- Fetch data from an external API using Axios
- Display API results with EJS
- Show an error message if the API request fails

## Technologies Used

- Node.js
- Express.js
- Axios
- EJS
- Body-parser
- CSS

## What I Practiced

- Making HTTP GET requests with Axios
- Using API query parameters
- Reading and using API documentation
- Handling different API response structures
- Rendering data with EJS
- Basic error handling
